# Instructions — MBS EPI–Financial Regression Pipeline

This document explains how to set up the environment, run the scripts, and reproduce all outputs in this repository.

## 1. Overview

This repository processes:

1. **EPI datasets** (topics, aspects, quarterly series)  
2. **Financial / MBS datasets**  
3. **Merged quarterly metrics**  
4. **Regression models** with results exported to CSV and an APA-style Word table  

The workflow is fully script-based and should be run from the repository root.

## 2. Repository contents

### Python scripts
- `EPI.py`  
  Loads and cleans EPI datasets.

- `Financial data.py`  
  Loads and prepares financial/MBS datasets.

- `MBS.py`  
  Merges EPI and financial data and builds quarterly metrics.

- `Regression.py`  
  Runs regression models and exports results.

### Input data files
- `epi_quarterly_true.csv`  
- `epi_topics.csv`  
- `epi_aspect_quarterly.csv`  

(Additional financial input files may be referenced inside `Financial data.py`.)

### Output files (generated by the pipeline)
- `mbs_quarterly_metrics.csv`  
- `mbs_merged_epi_financial.csv`  
- `regression_results_long.csv`  
- `apa_regression_table.docx`
